 研究简介与成果转化
本科毕业后创立科技公司，后于西安工程大学担任科研助理并攻读硕士，主攻智能无损检测技术，设计基于深度学习的超声检测、漏磁检测及视觉检测系统。成功研发智能无损检测机器人，实现焊缝跟踪、缺陷识别及壁面爬行等功能，相关技术已实现企业转化。目前作为主要成员承担省部级重点及厅局级项目7项，申请发明专利10余项，推动特种设备检测智能化升级。
